# Install
```bash
npm install
```

# About
The Cross-origin demos preview can be opened by running the following commands

## JSONP
```bash
npm run jsonp:node
npm run jsonp:web
```

## CORS
```bash
npm run cors:node
npm run cors:web
```

## document.domain
```bash
npm run domain
```

## window.name
```bash
npm run windowName
```

## postMessage
```bash
npm run postMessage:web1
npm run postMessage:web2
```

## proxy

```bash
npm run proxy:web
npm run proxy:node1
npm run proxy:node2
```
